How to get your API keys

Loopii needs Spotify API keys to sync your library. Follow these steps — it takes 2 minutes.

Go to the Spotify Developer Dashboard

Open your browser and go to the Spotify Developer Dashboard. Log in with your Spotify account (free or premium — both work).

Open Spotify Dashboard

Create a new app

Click the "Create app" button. Give it any name and description (e.g. "Loopii Player"). Set the redirect URI to:

http://localhost:8888/callback
http://localhost/callback

Check the Web API box under "APIs to use".

Copy your Client ID and Client Secret

After creating the app, you'll see the Client ID right on the dashboard. Click "Show client secret" to reveal the secret. Copy both.

Your keys are secret — never share them or commit them to GitHub.

Add the keys to the app

In the Loopii app folder, open or create the .env file and paste your keys:

SPOTIFY_CLIENT_ID=your_client_id_here
SPOTIFY_CLIENT_SECRET=your_client_secret_here

Save the file, then restart the app. That's it — you're done.

Troubleshooting

  • • Make sure the redirect URI in your Spotify app matches exactly what the app expects.
  • • If you get a "403" error, double-check your Client Secret is correct.
  • • You can always revoke app access from your Spotify account settings.
